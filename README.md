# 🪶 Sentinel’s Quill – Campus Journalism Website

## 📖 Overview

**Sentinel’s Quill** is a campus journalism website created to publish school news, feature stories, editorials, and announcements in an organized and accessible online platform.

This project aims to support student journalists by giving them a digital space to share their voices while reducing paper usage in school publications.

## 🎯 Purpose of the Project

This website was created as a **school project** to:

* Provide an online campus journalism platform
* Reduce paper-based publication costs
* Improve access to school news and announcements
* Help student journalists practice digital publishing

## ✨ Current Features

* 📰 News and Feature Article Pages
* 📚 Category-Based Content Sections
* 📷 Photo and Media Display
* 📱 Responsive Layout for Mobile and Desktop
* 🎨 Clean and Simple Campus Journalism Design

*(No login system or database yet – planned for future updates.)*

## 🚀 How to Run the Website

1. Download or clone the repository
2. Open the project folder
3. Double-click **index.html**
4. View the website in your browser

No installation needed because it’s a static website.
